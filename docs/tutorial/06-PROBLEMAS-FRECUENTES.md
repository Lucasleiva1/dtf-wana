# Problemas frecuentes

## Exportar está deshabilitado

Analice, resuelva alfa parcial y complete la revisión visual.

## Pulido dice que resuelva semitransparencias

Es una precondición técnica. Vuelva a Tratamiento de transparencias y deje cero valores intermedios.

## La limpieza no aplica

La máscara está vacía. Detecte automáticamente o seleccione manualmente al menos un píxel visible.

## Una imagen del lote tiene error

El mensaje queda en su tarjeta. Si alfa está desactivado y pulido está activo, active alfa o retire pulido para esa ejecución.

## La salida cambió a 8 bits

WebP y BMP fuerzan 8 bits. Use PNG o TIFF para conservar 16 bits.

## Se creó un nombre con número

Es la protección contra sobrescritura. Ya existía la ruta solicitada.

## Detuve el lote

El trabajo activo se cancela; no existe pausa. Los elementos pendientes permanecen para iniciar de nuevo.

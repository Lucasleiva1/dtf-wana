# Procesar por lote

1. Pulse **Por lote** para activar el fondo violeta.
2. Elija la carpeta de entrada. Se cargarán imágenes compatibles también desde subcarpetas.
3. Elija salida; si no lo hace, cada resultado queda junto a su original.
4. Elija PNG, WebP sin pérdida, TIFF o BMP. Configure PPP si usa PNG.
5. Active las etapas necesarias con **SÍ/NO**.
6. Ajuste cada panel una sola vez; la misma configuración se aplica a toda la cola.
7. Revise las miniaturas debajo del canvas. Arrastre horizontalmente o use las flechas.
8. Inicie. Cada éxito se exporta y desaparece; cada error queda visible.
9. Para interrumpir, use **Detener lote**. Lo pendiente se conserva.
10. Corrija los errores y reintente.

Consejo: si no sabe si las imágenes tienen alfa parcial, mantenga Transparencias activado antes de Residuos o Pulido.

# Procesar una imagen

## Abrir y analizar

Use **Abrir** o arrastre la imagen. El archivo original no se modifica. Pulse **Analizar alfa** para conocer cantidad y porcentaje de semitransparencias.

## Configurar

Puede elegir Conservador, Equilibrado o Agresivo, o mover el umbral. Active protecciones para conservar textura conectada, líneas finas y grunge. La reconstrucción automática calcula radio 1..4 según el tamaño; la manual acepta 1..16.

## Previsualizar

La preview se actualiza después de cambiar la configuración. Interprete rojo como transparente, cian como opaco y magenta como protegido/pendiente. No se modifica la imagen hasta aplicar.

## Aplicar y continuar

Pulse **Aplicar tratamiento**. Si el resultado ya es binario, puede limpiar residuos y luego pulir. Use comparar antes/después y deshacer si pierde detalle.

## Exportar

Complete la revisión visual y exporte. Si quedan semitransparencias, la aplicación bloquea la salida.

# Inicio rápido

1. Abra DTF Pro Studio.
2. Pulse **Abrir una imagen** y elija PNG, JPG, WebP, TIFF o BMP.
3. En **Tratamiento de transparencias**, pulse **Analizar alfa**.
4. Use la recomendación, previsualice y aplique.
5. Confirme que el análisis indique cero semitransparencias.
6. Revise el borde visualmente y pulse **Exportar**.

La salida individual se guarda como PNG y se reabre automáticamente para verificarla.

# Glosario

- **Alfa**: canal que representa transparencia.
- **Alfa binario/sólido**: cada píxel tiene alfa 0 o el máximo de su profundidad.
- **Semitransparencia**: alfa mayor que 0 y menor que 255/65535.
- **Región alfa**: componente conectado de 8 vecinos con alfa parcial.
- **Umbral**: valor que separa conversión a transparente de conversión a opaco.
- **Protección**: regla que impide volver transparente un detalle clasificado.
- **Pendiente**: píxel parcial que la configuración no resolvió.
- **Reconstrucción**: reemplazo condicionado del RGB al volver un píxel opaco.
- **Residuo**: componente o fragmento seleccionado para llevar a alfa 0.
- **Máscara**: selección binaria independiente de la imagen hasta aplicar.
- **Pulido**: modificación de una máscara alfa ya binaria para suavizar el borde.
- **Revisión**: versión monotónica del documento usada para evitar escrituras obsoletas.
- **Delta**: lista de píxeles antes/después usada por historial.
- **Job/trabajo**: operación asincrónica con progreso, resultado y cancelación.
- **Cola**: imágenes pendientes, en proceso o con error en modo lote.
- **Verificación de exportación**: reapertura de la salida y comprobación técnica.
- **MCP**: protocolo para exponer herramientas y recursos a una IA; no está implementado actualmente.
